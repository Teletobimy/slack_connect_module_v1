# Slack 채널별 담당자 메시지 리포트

오늘 하루 동안 각 채널에서 담당자별로 어떤 메시지를 보냈는지 분석하는 스크립트입니다.

## 기능

- 📡 모든 채널 목록 자동 수집
- 📅 오늘 날짜 메시지 필터링
- 👤 담당자별 메시지 그룹화
- 📊 채널별, 사용자별 리포트 생성
- 🖥️ 터미널에서 바로 확인 가능

## 설치

```bash
pip install -r requirements.txt
```

## 사용 방법

### 1. 환경 변수 설정

**Windows PowerShell:**
```powershell
$env:SLACK_USER_TOKEN="xoxp-your-user-token"
```

**영구 설정 (선택사항):**
```powershell
[System.Environment]::SetEnvironmentVariable('SLACK_USER_TOKEN', 'xoxp-your-user-token', 'User')
```

### 2. 스크립트 실행

```bash
python channel_report.py
```

## 주의사항

- **User Token 필요**: `xoxp-`로 시작하는 User Token이 필요합니다
- **권한**: User Token에 `channels:history`, `channels:read`, `groups:history` scope가 필요합니다
- **보안 채널**: 보안, admin 등 키워드가 포함된 채널은 자동으로 제외됩니다

## 환경 변수

- `SLACK_USER_TOKEN`: Slack User OAuth Token (필수)
